En vous appuyant sur un system message, l’outil de recherche web, le
mécanisme de mémoire, et le RAG, vous devez concevoir un agent intelligent jouant le rôle
d’un chef cuisinier personnel.


L’agent doit être capable de :


    recevoir la liste des ingrédients disponibles
    dans le réfrigérateur,
    mémoriser les préférences ou informations
    fournies par l’utilisateur (mémoire),
    utiliser un outil de recherche web ou RAG pour
    compléter ses connaissances culinaires si nécessaire (recettes,
    techniques, associations d’ingrédients), et proposer un ou plusieurs plats adaptés aux
    ingrédients disponibles.
